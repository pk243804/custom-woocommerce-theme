# Custom WooCommerce Theme

A **custom WordPress WooCommerce theme** designed for modern eCommerce websites.  
This theme is built with performance, clean UI, and WooCommerce customization in mind.

---

## Theme Overview

This custom WooCommerce theme provides a solid foundation for building online stores using **WordPress + WooCommerce**.  
It includes custom WooCommerce template overrides, responsive design, and easy customization.

---

## Features

- Fully compatible with WooCommerce
- Custom product listing & single product pages
- Responsive & mobile-friendly layout
- Custom header & footer
- Cart & checkout page styling
- Clean and optimized code
- Easy to customize
- SEO-friendly structure
